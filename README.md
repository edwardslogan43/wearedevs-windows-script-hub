## About WeAreDevs Executor

WeAreDevs Executor is a Windows desktop app built to run custom Lua scripts inside Roblox. It uses a simple one-click injection flow that attaches selected scripts to the Roblox client, supporting automation, gameplay tweaks, and general experimentation. The experience is intentionally straightforward, so it works well for both first-time users and more advanced scripters.

The app goes beyond basic execution by including an integrated script hub with more than 500 community-created scripts spanning different game types. It keeps resource usage low, does not need license keys, and includes an auto-update system to stay aligned with current Roblox changes. A multi-language UI and persistent script queue backed by SQLite round out the day-to-day workflow.

**Is WeAreDevs Executor safe to use?**
When you run scripts from third-party sources, responsibility rests with the user. The application itself is released under GPL v3.0, while the safety of scripts depends on where they come from.

**Will it work with the latest Roblox update?**
The auto-update engine is intended to keep the tool compatible with current Roblox releases. For the most recent status, check the repository release notes.

**How does this compare to other executors?**
Its main focus is staying lightweight while still offering a built-in hub with 500+ scripts and no key requirement, which makes it convenient for regular use.

**Can my Roblox account be affected?**
Any third-party execution tool comes with inherent risk. Be sure to review Roblox's terms of service and use the software at your own discretion.

**Where are my scripts stored?**
Imported scripts and queue information are saved locally in the `data/` and `scripts/user/` folders inside the application directory.
